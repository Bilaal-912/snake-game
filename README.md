🐍 Snake Game with Levels, Obstacles, Music, and Background

A classic Snake Game built using Python and Pygame, enhanced with background music, custom graphics, multiple levels, obstacles, score tracking, and increasing difficulty.

🎮 Features
Classic Snake gameplay
Background music support
Custom background image
Score tracking system
Three progressive levels
Screen wrapping in Levels 1 & 2
Obstacles introduced in higher levels
Boundary collision enabled in Level 3
Self-collision detection
Restart and Quit options after game over

🛠 Technologies Used
Python 3.x
Pygame

🚀 Installation
1. Clone the Repository
git clone https://github.com/yourusername/snake-game.git
cd snake-game
2. Install Dependencies
pip install pygame
3. Add Required Assets

Place the following files in the project directory:

megalovania.mp3
starfield.png
4. Run the Game
python snake_game.py
